
MODULES CUSTOM
---------------------------------

HEADER & FOOTER

This module allows using blocks in the header and footer that can be managed from the configuration "/admin/config"


TOUR 360

This module allows the configuration of points and navigation in a 360 environment, using an implementation of the "Marzipano" library, it also uses a configuration that can be managed in "/ admin / config"
